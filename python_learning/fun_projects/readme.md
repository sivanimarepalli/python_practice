# 🎮 Python Fun Projects

This folder contains small beginner-friendly projects built while learning Python.  
These programs helped me practice core programming concepts such as loops, conditions, functions, and user input.

---

## 📂 Projects Included

💰 **banking_prog.py**  
A simple banking system simulation that allows deposit, withdrawal, and balance checking.

🍿 **concession_stand.py**  
Simulates ordering items from a concession stand.

⏫ **count_up_timer.py**  
A timer program that counts upward.

⏳ **countdowntimer.py**  
A program that counts down to zero like a timer.

🎲 **dice_roll.py**  
Simulates rolling a dice using random numbers.

🔐 **encrypt.py**  
A basic text encryption and decryption program.

🪢 **hangman.py**  
Command-line version of the classic Hangman word guessing game.

📖 **madlibs_game_exercise.py**  
A Mad Libs style word game where users input words to create a funny sentence.

🎯 **no_guess_game.py**  
A number guessing game where the user guesses a randomly generated number.

❓ **quizgame.py**  
A simple quiz game that asks questions and tracks the score.

✊ **rock_paper_scissors.py**  
Classic Rock-Paper-Scissors game played against the computer.

🛒 **shoppingcartprog.py**  
Simulates a shopping cart where users add items and calculate the total cost.

🎰 **slot_machine.py**  
A slot machine game simulation using random symbols.

⏰ **working_alarm.py**  
A basic alarm clock program.
